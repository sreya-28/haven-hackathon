# HAVEN – Disaster Detection Prototype

HAVEN is a computer vision prototype developed during **Script Squad 2026** at Saintgits College of Engineering.

The system combines a custom CNN built with PyTorch and YOLOv8 object detection to identify potential disaster scenarios and estimate the number of people present in a scene. It supports image uploads, live camera monitoring, and threshold-based emergency alerts through a simple Tkinter interface.

## Technologies

* Python
* PyTorch
* OpenCV
* YOLOv8
* Tkinter

## Features

* Disaster vs Normal image classification
* Real-time camera monitoring
* People detection using YOLOv8
* Emergency alert triggering
* Image upload analysis

## Project Note

This project was developed as a hackathon prototype under strict time constraints. AI-assisted tools were used to accelerate development, debugging, and experimentation.
